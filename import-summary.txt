Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* AndroidManifest.xml => FujiSDK/src/main/AndroidManifest.xml
* res/ => FujiSDK/src/main/res
* src/ => FujiSDK/src/main/java/
* src/.DS_Store => FujiSDK/src/main/resources/.DS_Store
* src/com/.DS_Store => FujiSDK/src/main/resources/com/.DS_Store
* src/com/fuji/.DS_Store => FujiSDK/src/main/resources/com/fuji/.DS_Store
